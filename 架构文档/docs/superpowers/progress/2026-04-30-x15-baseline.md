## Codex review 修复

第一版 baseline (v1) 报 35→45 (+10)，Codex review 发现 Spring 域 markdown_anchor 漏 `#` 前缀（chunker AsciiDoc parser bug），导致 baseline 数据低估。

**修复**：commit `7661488` 在 ingestion API 层 `_row_to_metadata` 加 `_normalize_anchor` 兜底，6 个单测全 PASS。

**v2 实测验证**：47→60 (+13)，跟 Codex 预测完全吻合，Spring 域恢复正常的 12→15。

**Chunker 真 bug** 单独记 follow-up，不在本 plan scope。

**X1.5 通过 release gate，可上线**：
- 总召回率 +13/150 (+8.7pp)
